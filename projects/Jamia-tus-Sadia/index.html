Project Overview

JAMIA-TUS-SADIA is an international madrasha web project organized for modular development and clear documentation. The repository is structured to separate public pages, role-based pages, authentication, assets, scripts, data, and documentation.

Project Summary

This project uses a simple static site layout with progressive enhancement: Bootstrap for layout and responsiveness, modular folders for separation of concerns, and small JavaScript utilities for interactivity (navigation, collapsible folder tree, auth stubs).
