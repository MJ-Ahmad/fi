Project Overview

JAMIA-TUS-SADIA is an organized web project for an international madrasha. The repository separates public pages, role-based pages, authentication, assets, scripts, data, and documentation to support modular development and easy maintenance.

Primary goals: clear structure, responsive UI, role-based pages, simple auth flows, and maintainable assets.

Quick Facts
  • Owner: Fakhrul Islam
  • Start: Q1 2026
  • Stack: HTML, CSS (Bootstrap), JS
  • Structure: modular folders

Scope & Features

Core Features
  • Public pages: home, about, admission, gallery, contact
  • Role pages: admin, teacher, student, parent
  • Authentication stubs: login, register, password
  • Assets management: CSS, images, fonts
  • Client-side utilities: app.js, ui.js, dashboard.js
Non-functional Goals
  • Responsive layout using Bootstrap
  • Accessible markup and semantic HTML
  • Simple, maintainable folder layout
  • Progressive enhancement for interactivity
